# Acuity Analytics Challenge

This repository contains the solution to the Acuity Analytics Challenge project.

## Project Structure

| File / Folder           | Description |
|-------------------------|--------------|
| `Ascent_DS_Challenge_Data_set.csv`    | Contains input datasets used for analysis. |
| `data_analysis.ipynb`                 | Jupyter notebook used for data exploration and analysis. |
| `requirements.txt`      | Python dependencies required to run the project. |

## Requirements

The `requirements.txt` file lists all Python packages needed for this project.  
To install them, run:

```bash
pip install -r requirements.txt
